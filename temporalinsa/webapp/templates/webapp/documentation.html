Temporal'INSA - Documentation

Overview

Temporal'INSA is a web application designed for advanced time series classification and analysis. It provides researchers and data scientists with a flexible platform to experiment with various time series classification algorithms.

Key Features:
  • Multiple time series classification algorithms
  • Customizable experiment parameters
  • Support for UCR time series datasets
  • Comprehensive result visualization

Supported Algorithms

The platform currently supports the following time series classification algorithms:

  • Rocket: A fast and accurate time series classification method
  • MiniRocket: A more computationally efficient version of Rocket
  • L-Shapelets: Learn discriminative shapelets for classification
  • Hydra: A multi-head time series classification approach
  • Boss: Bag-of-SFA-Symbols method for time series classification

Supported Classifiers

The application offers two primary classifiers for final classification:

  • SVC (Support Vector Classifier):
    • Supports multiple kernel types (RBF, Linear, Polynomial, Sigmoid)
    • Customizable regularization parameter (C)
    • Flexible gamma settings
  • RidgeClassifierCV:
    • Cross-validated ridge regression classifier
    • Configurable alpha values
    • Multiple scoring metrics (Accuracy, F1, Precision, Recall)

Dataset Support

Currently, the platform supports the UCR Time Series Classification Archive datasets, including:

  • ADIAC
  • ACSF1
  • BEEF
  • CAR
  • CBF
  • ChlorineConcentration
  • CinCECGTorso
  • COFFEE
  • ECG200
  • ECG5000
Note: Custom dataset support is planned for future releases.

Experiment Workflow

  1. Name Your Experiment: Provide a descriptive name for tracking purposes
  2. Select Algorithms: Choose one or more time series classification algorithms
  3. Configure Parameters: Adjust algorithm-specific parameters
  4. Choose Classifier: Select and configure your preferred classifier
  5. Select Datasets: Pick one or more UCR datasets
  6. Start Experiment: Launch the classification process

Results Interpretation

After an experiment, you'll receive a detailed results page showing:

  • Experiment metadata (ID, name, date, datasets)
  • Performance metrics for each method
  • Raw classification results
